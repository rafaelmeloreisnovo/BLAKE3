# BugOrAdd — Documentação Absoluta (índice)

Este diretório agora está **estruturado por camadas** para evitar arquivos
soltos e facilitar reinterpretação contínua de conceitos.

## Estrutura oficial

- `models/` — formalização matemática e dinâmica do modelo.
- `lexicon/` — mapeamento entre línguas, prosódia e variação semântica.
- `experiments/` — protocolos reprodutíveis e trilha de validação.
- `notes/` — decisões de remodelagem e governança de leitura.

## Como usar

1. Começar em `models/toroidal_system_v1.md`.
2. Ler `lexicon/multilingual_signal_map.md` para contexto de tradução.
3. Executar/checkar critérios em `experiments/protocol_v1.md`.
4. Registrar decisões em `notes/restructuring_log.md`.

## Limite de escopo

Todo conteúdo aqui é externo/autoral (RMR) e **não** altera o núcleo
criptográfico do BLAKE3 upstream.
